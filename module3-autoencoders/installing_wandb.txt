The instructions below are for installing the wandb package 
on your AWS terminal.  

Also, I've included the list of reading material for today's 
assignment in this word document.

Installation details:
- Click on the "Git" button at the top, select "Git Command in Terminal"
- Type: source activate tensorflow_p36
- pip install --upgrade wandb
- go to wandb website (https://app.wandb.ai/settings), click on profile picture
- copy the API key
- back to terminal, type: wandb login YOURAPIKEY
